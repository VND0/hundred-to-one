Вопросы

Вопросы - это строительный блок игр. Создавайте вопросы, собирайте статистику о шести самых популярных ответов. Управление статистикой происходит вручную через интерфейс сайта.

Опросы

Для сбора статистики используйте систему опросов. Создайте опрос, добавьте вопросы и делитесь ссылкой для прохождения. Один вопрос может использоваться в нескольких опросах.

Игры

Для проведения игры требуется семь вопросов со статистикой: четыре - для раундов, три - для разыгровки первенства ответа. Вы управляете игрой самостоятельно, используя интерфейс сайта.
